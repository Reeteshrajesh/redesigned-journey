# Images Folder

This folder contains images used in the news application.

## Market Expansion Image
- File: `market-expansion-network.jpg`
- Description: Digital world map with network connections for the Market Expansion page hero section
- Recommended size: 1920x1080 or larger
- Format: JPG or PNG

To add the market expansion image:
1. Save your network world map image as `market-expansion-network.jpg` in this folder
2. The image will automatically be used as the background for the Market Expansion page hero section
